## Installation

To install the project and its dependencies, follow these steps:

1.  Ensure you have `npm` installed on your system. 

    ```sh-session
    npm install 
    ```

2.  Run the following command to install the project dependencies:
    ```sh-session
    cd frontend && npm install
    ```
